## Sobre mim

Sou apaixonado por transformar ideias em soluções digitais escaláveis. Atuo no desenvolvimento de sistemas com **Java e Spring Boot**, construção de **APIs REST**, integração com banco de dados, microsserviços, cloud e automação de processos.
